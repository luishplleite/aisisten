## Endpoint para Verificar se a Instância está Conectada

O endpoint para verificar o status de conexão de uma instância na Evolution API é:

### **GET** `/instance/connectionState/{instance}`

**Comando cURL:**

```bash
curl --request GET \
  --url https://seu-servidor.com/instance/connectionState/nome-da-instancia \
  --header 'apikey: sua-chave-api-aqui'
```

**Parâmetros:**[1][2]

- **Path Parameter (obrigatório):** `{instance}` - Nome da instância (não o ID)

**Resposta de sucesso (instância conectada):**

```json
{
  "instance": {
    "instanceName": "teste-docs",
    "state": "open"
  }
}
```

**Resposta quando desconectada:**

```json
{
  "instance": {
    "instanceName": "teste-docs",
    "state": "close"
  }
}
```

**Possíveis valores do campo `state`:**[2][1]

- **`open`** - Instância conectada ao WhatsApp
- **`close`** - Instância desconectada
- **`connecting`** - Instância em processo de conexão

**Observação importante:** Use sempre o **instanceName** (nome da instância), não o instanceId, ao fazer chamadas à API.[3]
