# Endpoints de Configuração de Comportamento da Instância

## 1. Atualizar Configurações (Definir Settings)

### **POST** `/settings/set`

Define ou atualiza as opções de comportamento da instância existente.[1][2]

**cURL:**
```bash
curl --request POST \
  --url https://seu-servidor.com/settings/set \
  --header 'apikey: sua-chave-api-aqui' \
  --header 'Content-Type: application/json' \
  --data '{
    "instanceName": "nome-da-instancia",
    "rejectCalls": true,
    "rejectCallsMessage": "Chamada rejeitada automaticamente.",
    "ignoreGroups": true,
    "alwaysOnline": true,
    "readMessages": true,
    "syncFullHistory": true,
    "readStatus": true
  }'
```

**Body (JSON):**  
- **instanceName** (string, obrigatório) – Nome da instância  
- **rejectCalls** (boolean) – Rejeita todas as chamadas recebidas  
- **rejectCallsMessage** (string, opcional) – Mensagem enviada ao rejeitar chamadas  
- **ignoreGroups** (boolean) – Ignora todas as mensagens de grupos  
- **alwaysOnline** (boolean) – Mantém o WhatsApp sempre online  
- **readMessages** (boolean) – Marca todas as mensagens como lidas  
- **syncFullHistory** (boolean) – Sincroniza todo o histórico de chat ao escanear o QR code  
- **readStatus** (boolean) – Marca todos os status como lidos  

**Resposta de sucesso:**  
```json
{
  "message": "Settings atualizadas com sucesso."
}
```

***

## 2. Obter Configurações Atuais (Buscar Settings)

### **GET** `/settings/get/{instance}`

Retorna o estado atual de todas as configurações da instância.[3][4]

**cURL:**
```bash
curl --request GET \
  --url https://seu-servidor.com/settings/get/nome-da-instancia \
  --header 'apikey: sua-chave-api-aqui'
```

**Path Parameter:**  
- **{instance}** (string, obrigatório) – Nome da instância  

**Resposta de sucesso:**  
```json
{
  "rejectCalls": true,
  "ignoreGroups": true,
  "alwaysOnline": true,
  "readMessages": true,
  "syncFullHistory": true,
  "readStatus": true
}
```

**Descrição dos campos na resposta:**  
- **rejectCalls** – Indica se chamadas são rejeitadas automaticamente  
- **ignoreGroups** – Indica se mensagens de grupos são ignoradas  
- **alwaysOnline** – Indica se o WhatsApp permanece sempre online  
- **readMessages** – Indica se mensagens são marcadas como lidas automaticamente  
- **syncFullHistory** – Indica se o histórico completo é sincronizado  
- **readStatus** – Indica se os status são marcados como lidos automaticamente
